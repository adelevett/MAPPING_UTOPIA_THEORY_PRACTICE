# Mapping Utopia: Theory & Practice (Frontend)

This directory contains the React + Vite frontend application for **Mapping Utopia**.

## What is this?
An interactive map connecting 803 community-led practices of resilience and reimagination with theoretical frameworks of educational utopianism. The frontend provides three views:
1. **The Map**: A geospatial visualization of practices colored by theoretical typology.
2. **Concept Atlas**: A deep-dive into the 40 categories of the underlying conceptual framework.
3. **About**: Project background and methodology details.

## Running Locally

To run the application locally, you need Node.js installed.

1. Navigate to the `frontend/` directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Data Requirements
The frontend expects compiled JSON data files to exist in `frontend/public/data/`. These files are generated by the python pipeline in the project root. If you are developing locally and the data is missing, run `python build_practices_json.py` in the root directory and copy the outputs to `public/data/` (as seen in `.github/workflows/deploy.yml`).

Required files:
- `practices.json`
- `category_index.json`
- `reference_index.json`
- `graph_edges.json`
- `coded_matrix.json`
- `references_resolved.json`
- `next_best_things.json`
- `ontology.json`
